O diagrama de blocos apresenta a estrutura geral do sistema embarcado, destacando a integração entre seus principais componentes. O ESP32 atua como unidade central, sendo responsável pela aquisição de dados do sensor de temperatura e umidade via barramento I2C, pelo armazenamento das informações em um módulo Micro SD por meio do SPI e pela comunicação com uma interface web via Wi-Fi. Além disso, o sistema conta com LEDs de sinalização controlados por GPIO, que indicam o estado de operação. A alimentação é fornecida por entrada USB, garantindo o funcionamento de todos os módulos. Dessa forma, o diagrama evidencia o fluxo de dados e a interação entre hardware e comunicação no sistema.
