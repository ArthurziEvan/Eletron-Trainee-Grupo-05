# Eletron-Trainee-Grupo-05

Repositório do Grupo 05 para o Processo Trainee da Eletronjun (UnB).

## Sobre

Este projeto tem como objetivo desenvolver um sistema de leitura de temperatura e umidade, com visualização dos dados em uma interface web e controle de modos de operação.

O sistema deverá:
- Ler dados de um sensor
- Exibir as informações em tempo real
- Permitir troca de modos (desligado, automático e ligado)
- Indicar o estado atual com LEDs
